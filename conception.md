# Flux

## Purpose

A terminal based tool used to record and display all your financial operations.

The first feature to come will be a general ledger which will allow the user to record and track his transactions aswell as the sum of all written transactions, which you might call his "balance".

The second feature will be a stock portfolio where the user can manage and track his investments.

The user actions will be made through the CLI with commands.

## Flux MVP

General ledger, add transactions to your ledger through the CLI, compute the ledger full history to get a fresh total of transactions in the terminal

* History first : Unique source of truth
* Append-only : History will never be modified
* Derived state, total amount is calculated, not stored

## Functionalities / User stories / Backlog

### MVP

1) Add a transaction
2) List all transactions
3) Calculate the users's balance
4) Display the users's balance
